# start timer

Adds a timer that counts UP to the top of the game window

```sig
startTimer()
```

## Parameters

none

## Example #example

In this example, we add a timer that counts up to the top of a window that has a purple background

```blocks

scene.setBackgroundColor(12)
//@highlight
carnival.startTimer()

```
